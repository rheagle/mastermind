# Mastermind

## To play this game:
### 1. Install Ruby
  **Mac:**
  ```bash
  brew install ruby
  ```
  **Windows:**
  ```bash
  ruby -v
  ```
### 2. Install Bundler and check that it is working
  ```bash
  gem install bundler
  ```
  ```bash
  bundler -v
  ```
### 3. Clone this repo
  ```bash
  git clone https://github.com/your-username/your-repo.git && cd your-repo
  ```
### 4. Install dependencies in the Gemfile.
  ```bash
  bundle install
  ```
### 5. Run the game!
  ```bash
  bin/start
  ```
